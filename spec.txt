Cabra

usuarios:
  nome (Char)
  hash (Char(255))  
 

contas: tipos = ["Bens", "Despesas", "Receitas", "Crédito", "Ajustes"]
  dono (FK) usuario
  tipo (Char) Um de "Bens", "Despesas", "Receitas", "Crédito", ou "Ajustes"
  sinal (Integer) 1 ou -1. "Bens" e "Despesas" são 1, o resto é -1
  nome (Char)
  orcamento (Integer)

  
transacoes:
  dono (FK) usuario
  data (Date)
  nome (Char) descrição
  valor (Integer) em centavos
  dr (FK) conta de débito 
  cr (FK) conta de crédito
  

Páginas:

[  ] editar_transacao.php?id=123&dr=M&cr=N: editar transacao
[  ] exec_editar_transacao.php
[  ] confirmar_excluir_transacao.php?id=123: Sim/Não hidden id
[  ] excluir_transacao.php: (POST) excluir somente a transacao

[  ] transacao.php?id=123: detalhes de uma transacao, editar, excluir

[  ] conta.php?id=123: saldo, link a extrato,
       lista de transacoes (paginar)

[  ] index.php: login.php ou exibir sumario se houver
       cookie ou sessao. Saldo de bens, credito, e ajustes
       (todas transacoes) e despesas e receitas deste mes

[  ] busca_transacao.php?q=ABC: navbar busca

[  ] extrato.php?conta=123&a=2018-01-01&b=2018-07-15: transacoes between
       dates a and b

[  ] ajuste.php?t=N: N é um dos "tipos de conta". Gerar automaticamente
       uma transacao para atualizar o saldo.
       Usar conta "Ajustes gerais",
       ver contas_util "get Contas apagadas or create it"
[  ] ajustar.php: exec, criar uma transacao automaticamente

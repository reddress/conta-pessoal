Cabra

usuarios:
  nome (Char)
  hash (Char(255))  
 

contas: tipos = ["Bens", "Despesas", "Receitas", "Crédito", "Ajustes"]
  dono (FK) usuario
  tipo (Char) Um de "Bens", "Despesas", "Receitas", "Crédito", ou "Ajustes"
  sinal (Integer) 1 ou -1. "Bens" e "Despesas" são 1, o resto é -1
  nome (Char)
  orcamento (Integer)

  
transacoes:
  dono (FK) usuario
  data (Date)
  nome (Char) descrição
  valor (Integer) em centavos
  dr (FK) conta de débito 
  cr (FK) conta de crédito
  

Páginas:

[  ] index.php: login.php ou exibir sumario se houver cookie ou sessao

[ok] tipo_de_conta.php?nome=N: N é um de
       "bens", "despesas", "receitas", "credito", ou "ajustes".
       Saldo de todas as contas do tipo selecionado.
       Exibir links caso nome não for especificado.

[  ] conta.php?id=123&a=2018-01-01&b=2018-07-15:
       extrato de transacoes entre datas a e b (opcionais)

[ok] nova_conta.php?tipo=N: N é um dos "tipos de conta",
       nome, e orcamento (0 se deixar em branco)
[ok] criar_conta.php: exec
[  ] editar_conta.php?id=123: semelhante a nova_conta.php
[  ] confirmar_excluir_conta.php: Sim/Não
[  ] excluir_conta.php?id=123: excluir todas as transacoes
       que a tiver como dr ou cr

[  ] transacao.php?id=123: detalhes de uma transacao

[  ] nova_transacao.php?dr=M&cr=N: M e N são um dos "tipos de conta". data, nome, valor, dr, cr
[  ] criar_transacao.php: exec
[  ] editar_transacao.php?id=123&dr=M&cr=N: editar transacao
[  ] confirmar_excluir_transacao.php: Sim/Não
[  ] excluir_transacao.php: excluir somente a transacao


Concluídas:

[ok] alterar_senha.php: exec
